# 🏴 Introduction to CTF Challenges

## 🔹 What is CTF?
- **Capture The Flag (CTF)** is a cybersecurity game.
- Goal: Solve puzzles and find hidden **flags** (secret strings).
- Think of it as a **digital treasure hunt**.

---

## 🔹 About This Lab Series
- Designed for **complete beginners**.
- No prior cybersecurity experience needed.
- Includes **step‑by‑step instructions**.
- Challenges are **basic and beginner‑friendly**.

---

## 🔹 The Basics You Need to Know
### 🏁 The Flag
- A flag is a secret string, e.g. `labex{flag_content_here}`.
- In this challenge: format is `labex-xxxx`.
- Finding the flag = proof you completed the challenge.

---

## 🔹 Common Challenge Types
| Type       | Description |
